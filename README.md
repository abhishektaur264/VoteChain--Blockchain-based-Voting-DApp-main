# 🗳️ VoteChain – Solana Blockchain Decentralized Voting DApp

A secure, transparent, and tamper-proof **Decentralized Voting Application** built on the **Solana blockchain**.  
This DApp ensures a **democratic, verifiable, and immutable** voting process powered by blockchain technology.

---

## 🚀 Overview

**VoteChain** allows users to participate in blockchain-based elections by connecting their **Solana wallet (Phantom, Solflare, etc.)**, registering as voters, and casting their votes directly on-chain.

Built with **Rust**, **Anchor Framework**, **Web3.js**, and **Next.js**, this project represents a complete **Full-Stack Web3 DApp** deployment on the Solana Devnet.

---

## ✨ Key Features

| Feature | Description |
|----------|-------------|

| 🧑‍🤝‍🧑 **Democratic Process** | Fair and transparent voting system with voter verification and admin approval. |
| 📊 **Real-time Results** | Track votes and outcomes instantly, ensuring full transparency. |

## 🛠️ Tech Stack

- **Blockchain:** Solana  
- **Smart Contracts:** Rust + Anchor Framework  
- **Frontend:** Next.js (React Framework)  
- **Blockchain Interaction:** Solana Web3.js  
- **Wallet:** Phantom Wallet

## 🧩 How It Works
### 🔹 Steps to Run Locally

```bash
# 1️⃣ Clone the repository
git clone https://github.com/<your-github-username>/votechain-dapp.git

# 2️⃣ Navigate to the project directory
cd votechain-dapp

# 3️⃣ Install dependencies
npm install

# 4️⃣ Run the Next.js development server
npm run dev

# 5️⃣ Open the app in browser
http://localhost:3000
````

---

## 🪙 Smart Contract Deployment

This project uses the **Anchor framework** for smart contract development.

```bash
# Build the Solana program
anchor build

# Deploy to Solana Devnet
anchor deploy
```

Once deployed, integrate your **program ID** into the frontend configuration.
